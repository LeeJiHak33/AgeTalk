발표 시나리오
- 개발동기
	최근 독거노인의 비율이 증가하고 있고 그에 따른 해결책인 노인 돌보미 로봇의 단점을 극복하기
	위해 Jetson Nano 기술을 활용한 채팅 서비스 계획하게 되었습니다.	
- 구조도 설명
	복지사가 관리하고 있는 어르신의 정보와 청년의 정보를 받고 어르신에게는 Jetson Nano를 제공하고 청년에게는
	매칭 연결을 도와줍니다
	어르신이 Jetson Nano를 통해 음성 데이터를 보내면 STT의 기능으로 텍스트로 변환되어 청년에게 전송되고
	청년이 사이트에 있는 채팅 기능에 텍스트를 입력하면 TTS의 기능으로 음성으로 변환되어 어르신의 Jetson Nano에
	출력됩니다.
- 메인페이지(채팅하기)
	버튼을 클릭하면 채팅창으로 이동하며 설문페이지를 작성하지 않았을 경우 얼럿이 출력됩니다
- 서브페이지(설문페이지)
	작성한 항목의 총점을 합산 후 유저의 우울도에 반영이 됩니다 반영된 우울도가 낮은 사람은 높은 사람과 매칭
	할 수 있도록 구현했습니다.
- 젯슨나노
	사용한 부품들은 아래와 같고 해당 부품들을 활용해 어르신들이 쉽고 편리하게 소통을 할 수 있도록 기능들을
	구현했습니다

- 개발환경 짧게

	젯슨나노의 기능을 구현할때에는 파이썬을 사용했고 나머지 사이트의 기능들은 아래의 기술을 사용했습니다
- 느낀점
	젯슨 나노를 처음 접해봐서 기자재를 찾는 부분이나 코드를 어떻게 구성해야 할지 DB와 연결은 어떻게 시킬지에 대해
	고민하는 부분이 어려웠습니다
	메인 기능적인 부분들인 jetson nano를 stt나 tts 기능들을 성공 시켰을 때 성장했다는 경험을 하였습니다
- 시연영상


짧은버전은 로그인 설문조사 매칭 기다림, 복지사가 매칭시켜줌 청년이 채팅 보냄 노인이 채팅 들음 채팅 보냄 청년이 채팅확인



유저

1. 회원가입
2. 로그인
3. 설문조사
4. 매칭을 기다려주세요
5. 매칭 후
6. qna 등록
7. 채팅창 띄우고 청년이 텍스트 입력
8. 노인으로 넘어가서(젯슨나노) 자막으로 청년이 보낸 채팅을 스피커를 통해 음성으로 들을 수 있다
9. 노인이 녹음
10. 청년 채팅(노인채팅 보여야함) 그걸 들음
 

복지사

1. 회원가입(현민이사진)
2. 성공 페이지
3. 승인 후 로그인 해서 매칭관리 페이지
4. 노인등록
5. 등록한 노인의 적절한 청년을 매칭 하는 장면
6. 부적절한 채팅을 확인 후 신고 후 매칭해제

관리자

1. 로그인
2. 복지사 승인 페이지
3. 첨부파일 확인 후 승인 버튼
4. 신고관리 리스트
5. 벤하는 과정
6. 정 넣을거면 벤당한 계정 로그인
